# Тестовое задание на позицию разработчик backend (Python)

Были созданы три эндпоинта:
1. Добавление новых устройств
2. Получение данных за все время по id устройства
3. Получение данных за определенный период по id устройства

## Запуск
Для запуска создайте файл .env в корне директории со следующим содержанием:
```
DB_NAME=
DB_HOST=
DB_PORT= 
DB_USER=
DB_PASS=
```
Запуск docker + docker-compose:
```
docker compose build
docker compose up
```
